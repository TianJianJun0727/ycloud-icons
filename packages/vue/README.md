# YCloud Icons Vue

Implementation of the YCloud icon library for Vue applications.

## Installation

```sh
pnpm add @ycloud-web/icons-vue
```

```sh
npm install @ycloud-web/icons-vue
```

```sh
yarn add @ycloud-web/icons-vue
```

```sh
bun add @ycloud-web/icons-vue
```
