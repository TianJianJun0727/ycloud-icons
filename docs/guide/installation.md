# Installation

## Web

Implementation of the YCloud icon library for web applications.

::: code-group

```sh [pnpm]
pnpm add ycloud@next
```

```sh [yarn]
yarn add ycloud@next
```

```sh [npm]
npm install ycloud@next
```

```sh [bun]
bun add ycloud@next
```

:::

For more details, see the [documentation](./ycloud/index.md).
